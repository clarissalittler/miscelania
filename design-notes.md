# Goals
This should be a minimalistic Android app that:

- Lets you write notes on your phone in either a rich text mode with bullets, italics, &c. or in straight markdown syntax.
  - There should be a toggle that lets you choose which mode you're writing in
- You can see the notes you've written in nesting directories, letting you group notes together
- You can choose color schema for the notes system
- You can embed images into documents
